# Modelo para Análisis Exploratorio de Datos en DataFrame de compensaciones ambientales

## LINK : https://icebreaker26.github.io/compensacionesambientales/

# Stack:
* Python
* Jupyter Notebook
* GH Pages

# Componentes:
- Panel de Control BI
- El nivel de cumplimiento físico (Siembra y Mantenimiento).
- Los requerimientos de flujo de caja proyectados hasta 2031.
- La eficiencia del gasto por individuo forestal y método de compensación.
- Distribución Geográfica de Proyectos
- Panel de Alertas y Riesgos de Incumplimiento
- Cronograma de Ejecución y Vencimiento Legal
- Análisis de Desviaciones Presupuestales (Outliers)
- Auditoría Automática de Mantenimientos (Text Mining)
- Segmentación Inteligente de Proyectos (Machine Learning)
- Modelo Predictivo de Presupuestación (Random Forest Regressor)
